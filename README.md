## 个人博客项目
  技术栈：koa2 + ejs + mysql 

### 项目描述
  个人学习koa2的练手项目。前端页面是从网上找的一份博客模板，使用ejs改成服务端渲染，富文本编辑器为了方便，选择了wangEditor.js 。基本上能满足使用
